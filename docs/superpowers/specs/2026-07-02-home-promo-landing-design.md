# Home 홍보 랜딩 재구성 설계

날짜: 2026-07-02

## 배경

`/` 루트의 `src/pages/Home.tsx`에 존재하지 않는 강사 3명이 하드코딩되어 있음.
이를 제거하고 학원(Hermes) 홍보용 풀폭 세로 스크롤 랜딩으로 교체한다.

## 요구사항

- 가짜 강사 소개 섹션(`members` 배열 + 카드 UI) 완전 삭제
- 전 섹션 `w-full`, 세로 스크롤 구성
- 이미지 파일/외부 이미지 생성 없이 React + Tailwind + 인라인 SVG만 사용
- 팔레트: 사이트 메인 톤에 맞춰 **화이트 베이스 + 블랙 강조** (모노크롬)
- 반응형 (모바일 1열 → 데스크톱 그리드)

## 섹션 구성 (위 → 아래)

1. **풀스크린 히어로** — 화이트 배경, `min-h-screen`. 초대형 블랙 타이포그래피
   캐치프레이즈 + 서브카피. CTA 버튼 2개: 블랙 솔리드(강좌 둘러보기 → `/courses`),
   블랙 아웃라인(강사진 보기 → `/instructors`). 하단 스크롤 유도 인디케이터.
2. **학원 소개 + 숫자 스탯** — 화이트. Hermes 소개 문단(체계적 커리큘럼·온라인
   강의·학습 관리) + 스탯 4개(누적 수강생, 만족도, 개설 강좌, 평균 경력 — 홍보용 수치).
3. **왜 우리 학원인가** — 블랙 반전 섹션. 강점 카드 4개(체계적 커리큘럼 / 온라인
   무제한 수강 / 1:1 질문·관리 / 학습 리포트). 아이콘은 인라인 SVG.
4. **수강생 후기 + 마무리 CTA** — 화이트. 후기 카드 3개(이니셜 아바타, 인물 사진
   없음) + 풀폭 CTA 배너(회원가입 → `/signup`, 강좌 보기 → `/courses`).

## 기술 결정

- `src/pages/Home.tsx` 단일 파일 재작성. 신규 의존성 없음.
- 스크롤 진입 시 섹션 fade-in: 라이브러리 없이 IntersectionObserver 훅 하나로 처리.
- `public/`의 강사 PNG(LeeJiEun.png 등)는 삭제하지 않음(다른 참조 여부 미확인).

## 검증

- `bun run build` (tsc + vite) 통과
- `/` 접속 시 강사 카드 없음, 4개 섹션 렌더, CTA 라우팅 정상
